# Portfólio Pessoal

Este é um projeto de portfólio pessoal desenvolvido com **React**, **TypeScript** e **Vite**. O objetivo do projeto é apresentar os projetos, habilidades e informações de contato de forma organizada e responsiva.

## 🛠 Tecnologias Utilizadas

- **React**: Biblioteca para construção de interfaces de usuário.
- **TypeScript**: Superset do JavaScript que adiciona tipagem estática.
- **Vite**: Ferramenta de build rápida para projetos modernos.
- **React Router**: Gerenciamento de rotas no React.
- **CSS**: Estilização customizada para o projeto.
- **Nginx**: Servidor web para hospedar a aplicação em produção.

---

## ✨ Funcionalidades

- **Página Inicial**: Apresentação pessoal e links para as seções principais.
- **Projetos**: Exibição de projetos com título, descrição, tecnologias utilizadas, links para o código e demonstração.
- **Contato**: Informações de contato e links para redes sociais.
- **Responsividade**: Layout adaptado para diferentes tamanhos de tela.

## 🚀 Instalação e Execução

### Pré-requisitos

- Node.js (versão 16 ou superior)
- Gerenciador de pacotes (npm ou yarn)

### Build da aplicação

- Basta entrar na pasta raiz do projeto e executar o arquivo de build utilizando o código:

```
   sh sh/build.sh
```

### Passos

1. Clone o repositório:
```
   git clone https://github.com/Flavio-LP/portifolio.git
   cd portifolio
   sh sh/build.sh
```
